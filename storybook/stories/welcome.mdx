# Welcome to the Component Library

This Storybook documents the reusable components, design foundations and interface patterns used across this website.

It provides a shared reference for designers, developers, content teams and accessibility reviewers. Components can be viewed independently, tested in different states and reused consistently throughout the Drupal theme.

## What you will find here

### Foundations

The visual building blocks used throughout the design system, including:

- Colours
- Typography
- Spacing
- Icons
- Other design tokens

### Components

Reusable interface elements organised using atomic design principles:

- **Atoms** — small foundational elements such as icons and buttons
- **Molecules** — combinations of atoms that perform a simple function
- **Organisms** — larger interface sections made from multiple components
- **Templates** — page-level layouts and structural examples

### Accessibility

Components should be designed and tested with accessibility in mind, including:

- Keyboard navigation
- Visible focus states
- Appropriate colour contrast
- Semantic HTML
- Screen reader support
- Responsive behaviour

## Using this Storybook

Use the navigation menu to explore the available foundations and components.

Each component page may include:

- Interactive controls
- Common variations
- Hover and focus states
- Accessibility guidance
- Implementation examples
- Drupal integration notes

## Project status

This component library is currently being developed as a proof of concept.

Components may change as the design system, Drupal theme and implementation approach continue to evolve.
